# for

for - это ключевое слово начинающее выполнение цикла. Циклы нужны для повторения определенной операции несколько раз.

Базовый цикл имеет 3 элемента:
- Начало
- Конец
- Шаг


```go
package main

import "fmt"

func main() {
	sum := 0
	for i := 0; i < 10; i++ {
		sum += i
	}
	fmt.Println(sum)
}

```

---

В го не обязательно заполнять все элементы цикла:

```go
package main

import "fmt"

func main() {
	sum := 1
	for ; sum < 1000; {
		sum += sum
	}
	fmt.Println(sum)
}

```
